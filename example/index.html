Participant Characteristic Explorer

Interactive data explorer with dynamic filtering by subgroups and characteristics of interest

Basic Explorer

This example features data collected across 20 years in 7 different Inner City Asthma Consortium studies.

Users click the buttons under "Subpopulation" to subset the data based on study, site, gender and age and then change the "Participant Characteristic" settings to get rates based on individual skin test results and the total number of positive tests for a participant. The "Summary" area on the left, which includes a simple visualization, is updated in real-time as buttons are clicked.

This deceptively simple chart allows a user to gain insight in to suprisingly complex issues with just a few clicks. For example, you can explore the following questions in this example: "What is the most common allergy in Cincinnati?", "How often is mouse allergy found in conjunction with dust mite allergy?", "Do females tend to grow out of cat allergies?", and "What is the most common allergy for participants who are mono-sensitized?"

Source Code

JavaScriptpopExplore.init(
	//element
	".example-viz",
	//data
	"../data/cross.csv",
	//settings
	{
		denominator:[{name:"study",head:"Study",type:"cat"},
					 {name:"site",head:"Site",type:"cat"},
					 {name:"gender",head:"Gender",type:"cat"},
					 {name:"age",head:"Age",type:"num"}
					],
		numerator:[{name:"atopy8",head:"Positive Skin Tests",type:"num"},
			{names: ["mouse_pos","dog_pos","derfderp_pos","cat_pos","germanroach_pos","rat_pos","alter_pos" ,"ragweed_pos"],
				labels:["Mouse"    ,"Dog"    ,"Dust Mite"   ,"Cat"    ,"German Roach"   ,"Rat"    ,"Alternaria","Ragweed"], 
				head:"Individual Skin Tests",
				type:"checklist", positive:["1"]
				}
				  ],
		viz: true
	}
);

Data Download

The tool is generated based on a comma-delimited dataset (.CSV file type) where each row contains information for an individual participant. Any column in the dataset can be used to create a control in the explorer. For full details, see the Technical Details section in the User Guide.
